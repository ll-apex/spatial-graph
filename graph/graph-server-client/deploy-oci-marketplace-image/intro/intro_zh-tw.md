# 簡介

## 關於此研討會

本研討會中的實驗室會逐步介紹部署、設定及使用 Graph Server 和從屬端套件，以查詢、分析及視覺化方式呈現儲存在 Free Tier Autonomous Database (ADB) 執行處理中的圖表內容。您將佈建新的免費層 ADB 執行處理、將資料載入其中、建立圖表，然後查詢、分析及視覺化圖表。

預估研討會時間：1 小時

### 關於 Oracle Property Graph？

Oracle Database 特性圖表支援並包含圖形儲存、索引和搜尋；這是一個功能強大的記憶體內分析師，擁有近 60 個內建的平行分析功能；一個圖形查詢語言和開發人員 API。圖形演算法可讓您探索和發現社群網路、IoT、大數據、資料倉儲和複雜交易資料中的關係，例如銀行業務詐騙偵測、客戶 360 和智慧型製造。

### 目標

在此研討會中，您將：

*   佈建免費層運算與 ADB 共用執行處理。
*   部署 Graph Server 與 Client Marketplace 影像。
*   在 ADB 執行處理中建立及填入表格。
*   以圖表形式查詢、分析及視覺化該資料。

本研習適用於 Graph Server 版本 **21.4** 測試。
