# 如何建立 OCI 區間？

期間：2 分鐘

區間可讓您組織與控制雲端資源的存取權。

### 先決條件

*   一個 Oracle Cloud Account。

## 作業 1：建立 OCI 區間

1.  如果您尚未登入，請使用其他瀏覽器頁籤或視窗登入您的 Oracle Cloud 帳戶。
    
2.  開啟**瀏覽**功能表，然後按一下**識別與安全**。在**識別**底下，按一下**區間**。
    
3.  在**區間**頁面中，按一下**建立區間**。
    
4.  在**建立區間**對話方塊的**名稱**欄位中輸入區間的名稱，在**描述**欄位中輸入選擇性的描述。
    
5.  在**父項區間**下拉式清單中，選取您的父項區間，然後按一下**建立區間**。
    
    ![在已完成的「建立區間」對話方塊中，按一下「建立區間」。](./images/create-compartment.png " ")
    
    **區間**頁面會重新顯示，而新建立的區間則會顯示在可用區間清單中。
    
    ![新建立的區間會醒目提示其狀態為「作用中」。](./images/compartment-created.png " ")
